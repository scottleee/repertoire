Google Drive Sync

Your repertoire will be saved as repertoire-data.json in your Google Drive. One-time setup (~5 min).

  1. Go to console.cloud.google.com and sign in.

  2. Click Select a project → New Project. Name it anything and click Create.

  3. Go to APIs & Services → Library. Search for Google Drive API and click Enable.

  4. Go to APIs & Services → OAuth consent screen. Choose External, fill in your email, save through all steps. Add your email as a test user.

  5. Go to Credentials → Create Credentials → OAuth client ID. Choose Web application. Under Authorised JavaScript origins add null. Click Create.

  6. Copy the Client ID (ends in .apps.googleusercontent.com) and paste below.

✓ Connected as

Data syncs automatically on every change to repertoire-data.json in your Drive.
